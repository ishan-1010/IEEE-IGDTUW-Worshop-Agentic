Teaching

Why these three?

Structured output — APIs and code need data, not prose. "Return ONLY valid JSON" → your code gets a dict; no regex.

Tools — The model doesn't have your data. Tools = eyes (read docs, APIs) and hands (run code). Model decides when to call them.

Self-correction — First draft is rarely production-ready. Reflection: generate → critique (find problems) → fix. Same in Devin, AutoGPT.
